# Mock Item Manager MERN Starter Project

This is a practice project for the SE2020 WMT Lab Test.

## Lab Task
Add a new field called **Brand Name**.

You must update:
1. Backend model
2. Backend route/controller
3. Frontend form
4. POST request body
5. Home page display

## Run Backend
```bash
cd backend
npm install
copy .env.example .env
# Add your MongoDB Atlas URL to .env
npm run dev
```

## Run Frontend
```bash
cd frontend
npm install
npm run dev
```

Frontend: http://localhost:5173  
Backend: http://localhost:5000
